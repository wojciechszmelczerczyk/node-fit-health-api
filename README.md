## To run 

#### Clone repository

```
git clone https://github.com/wojciechszmelczerczyk/node-fit-health-api.git
```

#### Navigate to project

```sh
cd ./node-fit-health-api
```

#### Install dependencies

```
npm i
```

#### Run Express server

```
npm run start
```
